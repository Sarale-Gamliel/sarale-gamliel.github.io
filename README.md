# Sarale Gamliel - Portfolio

Personal portfolio site for Sarale Gamliel, AI Product Developer and Full Stack Developer. Built as a static site with vanilla HTML, CSS, and JavaScript, deployed on GitHub Pages.

Live site: https://sarale-gamliel.github.io

## What is on this site

An about section with skills and experience, a projects section that links out to live demos and source code for CLIQ Trivia, the Games Arcade, client landing pages, and other work in progress, plus contact details and links to LinkedIn and GitHub.

## Tech stack

HTML5, CSS3 (custom design tokens, no framework), vanilla JavaScript, and GitHub Pages for hosting.

## Project structure

The assets folder holds images and static files, the css folder holds stylesheets, the js folder holds the scripts that render project cards and page interactions, and the tests folder holds front-end tests.

## Getting started

This is a static site, so it can be opened directly in a browser, or served locally with any static file server.

## Roadmap

Planned improvements include richer project showcases with device mockups, an animated skills section, a timeline of experience, and a GitHub activity widget.
